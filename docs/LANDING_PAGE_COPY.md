## COPY TONE GUIDELINES

### For Traders (They Respond To):
✓ **Specificity over hype** — "0-second delay" not "lightning fast"
✓ **Edge, not entertainment** — This is a tool, not a vibe
✓ **Proof, not promises** — "15,000+ hours analyzed daily" not "best platform ever"
✓ **Speed + clarity** — Traders are busy. Respect that.
✓ **Trust through transparency** — Own the limitations. What's in beta? Say it.

### Avoid:
✗ Emojis in headers (except feature icons)
✗ Salesy language ("life-changing," "revolutionary")
✗ Comparisons to competitors (it screams weakness)
✗ Testimonials that sound fake
✗ Dark patterns (hidden pricing, forced signups)

---

## CONVERSION NOTES

1. **CTA above fold** — "Get Early Access (Free)" should appear in hero
2. **Secondary CTAs** — Repeat signup form before footer (mid-page scroll)
3. **Trust early** — Data sources + uptime in hero or problem section, not buried
4. **Reduce friction** — No credit card for beta. One-click signup if possible.
5. **Clear next step** — What happens after they click? Set expectations immediately.
6. **Mobile-first** — Traders check on phones between trades. Stack vertically.
