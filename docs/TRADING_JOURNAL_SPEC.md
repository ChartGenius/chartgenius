# Trading Journal Feature Spec
## Reference: Tradezella ($29-49/mo)

### Core Features to Build

#### Trade Logging
- Entry/exit price, date/time
- Position size (shares, contracts, lots)
- Asset class (stock, option, futures, forex, crypto)
- Direction (long/short)
- Commissions/fees
- P&L (auto-calculated)
- R-Multiple (actual risk vs reward achieved)

#### Tags & Categories
- Setup tags (breakout, pullback, reversal, trend, etc.)
- Mistake tags (FOMO, oversize, early exit, revenge trade, etc.)
- Custom user-defined tags
- Strategy categories

#### Analytics Dashboard
- Win rate, loss rate
- Average win $, Average loss $
- Profit factor (gross profit / gross loss)
- R-Multiple average
- Expectancy (avg win × win rate - avg loss × loss rate)
- Best/worst trading day
- Win/loss streaks
- Consecutive win/loss tracking

#### Calendar View
- Monthly grid showing green (profit) / red (loss) days
- Click day to see all trades
- Daily P&L totals
- Weekly/monthly summaries

#### P&L Charts
- Cumulative P&L curve
- Daily P&L bar chart
- Weekly/monthly breakdown
- By asset class
- By setup type
- By time of day

#### Advanced Filtering
- Date range
- Asset class
- Setup type
- Win/loss
- Tag
- Minimum P&L threshold
- Symbol

#### Trade Notes & Screenshots
- Per-trade notes field
- Screenshot upload (chart at time of trade)
- Pre-trade plan vs actual execution
- Lessons learned per trade

#### Notebook / Playbooks
- Trading plan templates
- Strategy documentation
- Daily/weekly recap templates
- Custom templates

#### Statistics & Reports
- MAE (Max Adverse Excursion) — biggest drawdown during trade
- MFE (Max Favorable Excursion) — biggest unrealized gain during trade
- Average hold time
- Best/worst hour, day, setup
- Performance by day of week
- Performance by time of day
- Drawdown analysis

#### Multi-Asset Support
- Stocks: shares, price
- Options: contracts, strike, expiry, premium
- Futures: contracts, tick value
- Forex: lots, pips
- Crypto: coins, price
